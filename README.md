# SQL Sales Data Analysis Project

This project analyzes sales data using SQL queries.

## Tools Used
- SQL

## Dataset Columns
- sale_id
- product_name
- category
- quantity
- price
- sale_date

## Analysis Performed
1. Total Sales Revenue
2. Sales by Product
3. Revenue by Category
4. Top Selling Product
5. Monthly Sales Analysis

## Purpose
This project helps understand sales trends, product performance, and revenue insights using SQL.
